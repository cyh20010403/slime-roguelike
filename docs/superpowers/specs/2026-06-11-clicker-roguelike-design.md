# 点击挂机 Roguelike 游戏设计文档

> 创建日期：2026-06-11 | 状态：已确认

---

## 一、项目概述

一款桌面 Web 横屏的点击挂机 Roguelike 游戏。可爱休闲画风，高频轻选的构筑节奏，玩家通过点击屏幕击败不断出现的敌人，存活尽可能久的时间。

### 核心目标
- 轻度玩法，爽快体验，吸引泛用户
- 数小时的游戏时长和 replayability
- 组件化解耦架构，便于迭代和扩展

---

## 二、技术选型

| 项目 | 选择 |
|------|------|
| 渲染方案 | Canvas 游戏层 + DOM UI 层（方案 C） |
| 技术栈 | 原生 HTML/CSS/JS，零依赖 |
| 平台 | 桌面 Web 横屏（1920×1080） |
| 素材方案 | 全代码生成（CSS/SVG/Canvas/Web Audio），零外部文件 |
| 美术风格 | 可爱休闲风，Q版卡通，明亮色彩 |

## 四、核心战斗系统

### 4.1 点击交互
- 点击屏幕任意位置，对该位置周围半径 80px 内的所有敌人造成伤害
- 点击范围随升级可扩展
- 反馈链：波纹扩散 → 伤害数字弹出 → 击退微移 → 粒子特效

### 4.2 敌人系统

| 参数 | 说明 |
|------|------|
| 出现方式 | 屏幕边缘随机位置生成 |
| 移动方式 | 缓速向屏幕中心/随机方向移动 |
| 存活时间 | 普通 8-15s，Boss 30-45s |
| 超时惩罚 | 敌人超时未击杀，对玩家造成 5-15 HP 伤害 |
| 死亡奖励 | 掉落金币，稀有怪掉落大量金币 |

### 4.3 伤害流程
```
点击 → 检测范围内敌人 → 计算伤害（ATK×暴击×元素加成）→
扣血 + 弹出数字 + 击退微移 + 粒子特效 → 敌人死亡 → 金币掉落 + 消除动画
```

### 4.4 Boss 战
- 每 5 波普通敌人后出现一个 Boss
- Boss 从屏幕中央出现，体型 200-300px，有独立大血条
- Boss 期间普通敌人继续出现但频率降低
- 击败后暂停游戏，弹出 3 选 1 升级面板

### 4.5 玩家属性（初始值）

| 属性 | 初始值 | 说明 |
|------|--------|------|
| 攻击力 (ATK) | 10 | 每次点击基础伤害 |
| 点击范围 | 80px | 圆形 AOE 半径 |
| 暴击率 | 5% | 暴击 ×2 伤害 |
| HP | 100 | 归零游戏结束 |
| 金币加成 | 0% | 额外金币百分比 |

## 五、升级卡片系统

### 5.1 触发机制
- 击败 Boss → 暂停游戏 → 弹出 3 选 1 面板
- 同轮至少保证 1 张稀有+

### 5.2 稀有度

| 稀有度 | 概率 | 边框色 | 特点 |
|--------|------|--------|------|
| 普通 ⭐ | 50% | 灰绿 | 基础数值提升 |
| 稀有 ⭐⭐ | 30% | 蓝紫 | 特效/机制变化 |
| 传说 ⭐⭐⭐ | 15% | 金橙 | 强力质变 |
| 神话 ⭐⭐⭐⭐ | 5% | 彩虹 | 颠覆性改变 |

### 5.3 四大品类

**属性强化**
- 攻击力 +10/25/50/100%
- 攻速（点击冷却缩短）
- 暴击率 +5/10/20%
- 暴击伤害 +50/100/200%
- 点击范围 +20/40/80px
- 金币加成 +30/60/100%

**攻击特效**
- 火焰附魔：点击附加燃烧（3秒 DOT）
- 冰冻之触：点击减速敌人 50%，持续 2s
- 闪电链：点击伤害跳跃至附近 3 个敌人
- 分裂弹：每次点击射出 4 枚飞弹
- 死亡新星：敌人死亡时爆炸，AOE 伤害

**召唤随从**
- 小史莱姆：自动攻击最近敌人（可叠加数量）
- 火焰精灵：远程火球，带灼烧
- 护盾精灵：环绕玩家，碰撞敌人造成伤害
- 金币史莱姆：击杀敌人额外掉落金币
- 治疗妖精：每 10s 回复 5 HP

**被动光环**
- 伤害光环：每 5s 对全屏敌人造成伤害
- 回复光环：每 8s 回复 3 HP
- 磁铁光环：金币自动飞向玩家
- 时间减速：敌人存活时间延长 30%
- 荆棘光环：敌人造成伤害时反弹 50%

### 5.4 构筑上限
- 每种卡可重复获取，效果叠加（线性或递减）
- 同卡第 3 次后出现概率递减，鼓励混合构筑

## 六、波次 & 难度曲线

### 6.1 波次结构
```
波次 1-4  → 普通敌人，逐步增加数量和速度
波次 5   → Boss #1
波次 6-9  → 新敌人类型解锁，属性提升
波次 10  → Boss #2
波次 11-14 → 引入精英怪（稀有，掉落更好）
波次 15  → Boss #3
...依此类推，难度持续攀升
```

### 6.2 敌人参数表

| 类型 | 出现波次 | HP | 速度 | 存活 | 伤害 | 金币 | 体型 |
|------|---------|-----|------|------|------|------|------|
| 绿色史莱姆 | 1 | 10 | 慢 | 12s | 5 | 5-10 | 40px |
| 蓝色史莱姆 | 3 | 25 | 中 | 10s | 8 | 10-15 | 48px |
| 小恶魔 | 6 | 40 | 快 | 8s | 10 | 15-20 | 36px |
| 红色史莱姆 | 10 | 80 | 慢 | 15s | 15 | 20-30 | 60px |
| 暗影恶魔 | 14 | 120 | 极快 | 7s | 12 | 25-35 | 36px |
| 精英⭐ | 12+ | 200 | 中 | 10s | 20 | 50-100 | 52px |

- Boss HP = 普通敌人 × 15-25 倍
- 每 5 波，所有敌人属性 ×1.3

### 6.3 预期生存时长

| 构筑水平 | 到达波次 | 存活时间 |
|----------|---------|---------|
| 新手 | 6-8 | 4-6 分钟 |
| 有意识 | 10-12 | 8-10 分钟 |
| 成型构筑 | 15-18 | 12-15 分钟 |
| 天胡 | 20+ | 18+ 分钟 |

- **Boss 血条**：屏幕顶部大血条，Boss 名称 + HP 数值
- **波次提示**：大文字飘入飘出，"BOSS 来袭!" 红色警告
- **游戏结束面板**：存活时间、击杀数、构筑回顾、"再来一局" 按钮

## 八、视觉风格

### 8.1 色彩方案
- 主色调：奶油背景 #FFF8E7 + 粉红主色 #FF6B8A + 金黄强调 #FFD93D
- 辅助色：天蓝 #6BC9FF、薄荷绿 #A8E6CF
- 整体明亮、温暖、愉悦

### 8.2 角色设计
- Q版圆润造型，大眼睛，呆萌表情
- 史莱姆（弹弹果冻）、小恶魔（毛球+翅膀）、胖龙（圆滚滚 Boss）
- 被攻击时变形（压扁、眨眼、鼓起腮帮子）

### 8.3 素材生成策略
所有素材通过代码程序化生成，零外部文件依赖：
- 敌人：CSS 绘制 + Canvas 渲染
- 特效：Canvas 程序化粒子
- UI 图标：CSS + emoji + SVG
- 音效：Web Audio API 合成

## 九、音效系统

### 9.1 技术方案
Web Audio API 程序化合成（OscillatorNode + GainNode + Noise），零音频文件。

### 9.2 音效清单

| 类别 | 音效 | 合成方式 |
|------|------|---------|
| 战斗 | 点击命中 | 低频脉冲 sine 80Hz 50ms |
| | 暴击 | 命中音升高 + 高频泛音 |
| | 击杀 | 清脆 pop triangle 400→200Hz |
| | Boss 命中 | 低沉重击 sine 60Hz |
| | Boss 击杀 | 爆裂 noise burst + 低频余韵 |
| 反馈 | 金币拾取 | 叮叮声 sine 800→1200Hz |
| | 升级选择 | C-E-G 分解和弦 |
| | 波次提示 | 号角 sawtooth 上扬 |
| | Boss 警告 | 低沉警报 square 150Hz |
| 状态 | 玩家受伤 | 下降音 sine 300→150Hz |
| | 游戏结束 | 下行小旋律 triangle |
| | HP 过低 | 心跳 thump |
| 氛围 | 背景 BGM | 多轨道循环 8-16 小节 |
| | Boss BGM | 加速 + 低音增强 |

### 9.3 混音
- 全局音量 / 音效 / 音乐独立通道
- 音效优先级队列，同时最多 8 个

## 十、核心设计原则

- 每个 JS 文件职责单一，最长不超过 400 行
- Canvas 层只做渲染，UI 交互由 DOM 层处理
- 升级效果通过注册式系统扩展（新增效果不修改核心代码）
- 敌人/波次数据驱动（JSON 配置）
- 游戏存档：浏览器 localStorage 保存最高分和统计
